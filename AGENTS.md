# AGENTS.md

The agent instructions for this repository live in **[`CLAUDE.md`](CLAUDE.md)** — one
canonical copy, regardless of which tool or model you are. Read it in full before
working here; it carries the build/test commands, the WSL↔Windows quirks, the
architecture map, the strict-TDD contract, and the hard safety/IP boundaries.

The two rules you must know even before opening it:

- **Never** execute downloaded installers, install drivers, write the live registry,
  or delete recursively. No exceptions without a named, owner-signed register entry.
- **Never merge** — every change is one branch, one PR; the owner merges.

Process law is `CONTRIBUTING.md`. The governing docs chain and everything else:
`CLAUDE.md`.
